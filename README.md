# AIML_dca_classifier
Berkeley AIML Practical Application 1-  python project to  predict whether drivers redeem coupons based on contextual features using a UCI survey dataset.

This project focuses on predicting coupon acceptance among drivers who visit bars. Using survey-based sample data collected from a diverse set of drivers, the analysis aims to uncover the key factors that most strongly influence whether a driver accepts a bar-related coupon.

**Notebook Link**
prompt.ipynb

**Key Features Considered for the analysis**
Age
Income
Passengers
Frequency of visits to the Bar

**Key Findings**
Coupon acceptance is high among drivers who traveled with friends.
Coupon acceptance has a positive regression with the number of visits.
Age also plays an important role in the acceptance of the coupons. higher the age people accepted the coupons more.
